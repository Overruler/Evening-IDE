Mylyn WikiText User Guide
Getting Started

Overview

This guide provides basic instructions on the use of WikiText.
WikiText provides extensions to the Mylyn task editor supporting lightweight markup languages such as Textile and MediaWiki. WikiText also provides a wiki text editor for Eclipse, a Maven plug-ins and Ant tasks for generating from lightweight markup to HTML and other formats.

For information about integrating your application with WikiText please see the WikiText Developer Guide.
