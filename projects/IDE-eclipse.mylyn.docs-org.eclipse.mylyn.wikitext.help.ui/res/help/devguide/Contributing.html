Contributing

WikiText is evolving as a direct result of user contributions. Please feel free to contribute by

  • posting a bug
  • submitting a patch

All bugs and patches are tracked via Bugzilla. See Issue Tracking below for more details.

For more information about contributing please see the Mylyn Contributor Reference.

Issue Tracking

Issues are tracked in Eclipse Bugzilla under the Mylyn Docs project, Wikitext component. Please feel free to submit bugs, enhancement requests and patches.
